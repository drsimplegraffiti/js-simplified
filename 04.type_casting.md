#### Type Casting in JavaScript

Type casting is the process of converting a value from one data type to another (such as string to number, object to boolean, and so on).

#### Examples of Type Casting

##### String to Number

```js
var a = '14';
var b = '2';
var c = a / b;
console.log(c); // 7
```

##### Number to String

```js
var a = 14;
var b = 2;
var c = a / b;
console.log(c); // '7'
```
